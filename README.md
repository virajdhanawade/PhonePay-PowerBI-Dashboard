# PhonePe Transaction Analysis Dashboard

## Overview

This project is an interactive Power BI dashboard built to analyze PhonePe transaction data and generate business insights.

## Tools Used

 Power BI
 SQL
 DAX

## Key Features

 Total Transactions KPI
 Total Transaction Value Analysis
 Success Rate Monitoring
 User Demographics Analysis
 Age Group Analysis
 Service Category Performance
 Dynamic Filters and Slicers
 Business Insights using DAX
